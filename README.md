# proom
picture room

# usage
```html
  <!-- proom repo -->
  <meta name="proom:repo" content="./gallery.js" />
  <!-- proom property -->
  <script type="text/javascript" src="proom.js"></script
```
initialize onload
```html
<body onload="proom.init()">
```
and this is sample 
